# 🧮 Math Calculator

A simple and responsive mathematical calculator built using **HTML**, **CSS**, and **JavaScript**. It performs basic arithmetic operations with a clean and user-friendly interface.

🔗 https://math-calculator-beta.vercel.app

- ➕ Addition

## 💻 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/Dhivakar2004/Math-calculator.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
